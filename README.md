# Suppliers Database Practice

This repository contains practice work for building and managing a PostgreSQL database using Python and psycopg2.

## Purpose

The goal of this project is to strengthen my understanding of:

- Relational database design
- PostgreSQL schema creation
- Table relationships and constraints
- Python database connectivity using `psycopg2`
- Executing SQL commands programmatically
- Managing database projects with proper Git workflow

## Technologies Used

- Python 3
- PostgreSQL
- psycopg2
- Git & GitHub

## Structure

- SQL table creation scripts
- Python scripts for database connection and table setup
- Practice queries and data manipulation examples

## Learning Focus

This repository is focused on:

- Proper database initialization
- Writing clean SQL DDL statements
- Understanding primary keys and foreign key constraints
- Handling connection management and transactions in Python
- Maintaining clean repository structure and version control discipline

This is a practice repository used to reinforce database fundamentals and backend development skills.
